Glen Pi Geeks

From 2017 to 2021, I was apart of a self-run group with five friends known as the Glen Pi Geeks. We had 2-3hr weekly meetings researching and programming projects. Our largest project was known as the Photobooth in which we were able to successfully create a program that could recognize a person's emotions as well as recognize a face and match it to a name. Some of our other projects include:

  • Marshall Financial Group Project
  • Stock Market Analysis Project
  • Portfolio Optimization and Analysis
  • Box Office Data Analysis Project
  • Covid-19 Data Analysis Project
